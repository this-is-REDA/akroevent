# akroevent

Site web officiel **Akro Event** — agence événementielle marocaine 360° (team building, événementiel corporate, stands, gestion déléguée).

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase

## Développement

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Variables d'environnement

Copier `.env.local.example` vers `.env.local` et renseigner les clés Supabase et l'URL du site.
